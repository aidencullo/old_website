Description

I've been trying to utilize native mac keyboard shortcuts to speed up app navigation and manipulation. I'll share here the most pertinent and useful shortcuts i've come across.

I want to look more into navigating through apps as right now i really just tab around, but i know there are easier/quicker ways.

Chrome

Chrome is my primary browser and I really want to navigate it easier, i always find that I want to highlight the search bar or some element without having to use the mouse or tab a million times. I also really don't want to use the arrow keys either cause it's slower.

  1. ⌘ + ⌥ + i
    2. Inspect/Close source
  2. ⌘ + ↑
    3. Page up
  3. ⌘ + ↓
    4. Page down
  4. ⌘ + ←
    5. Page back
  5. ⌘ + →
    6. Page forward

Any Application

  1. ⌘ + ⌃ + f
    2. Maximize/minimize
  2. ⌘ + ,
    3. Preferences

Terminal

  1. ⌃ + u
    2. delete command line
  2. ⌃ + w
    3. delete word on command line
